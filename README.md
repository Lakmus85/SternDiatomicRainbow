# Stern Diatomic Rainbow
This project is visualizing the [Stern's Diatomic Sequence](https://en.wikipedia.org/wiki/Calkin%E2%80%93Wilf_tree#Stern's_diatomic_sequence) as a bar chart in rainbow colors.
